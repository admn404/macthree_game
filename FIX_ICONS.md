# Исправление ошибок 404 для иконок

## Проблема
Сервер выдает ошибки 404 для `icon-192.png` и `favicon.ico`. Это не критично, но лучше исправить.

## Решение 1: Создать иконки через Python (если установлен Pillow)

```bash
cd MacThree-Web
python create_icons.py
```

Если Pillow не установлен:
```bash
pip install Pillow
python create_icons.py
```

## Решение 2: Создать иконки через браузер

1. Откройте файл `create_icons_simple.html` в браузере
2. Нажмите кнопку "Создать иконки"
3. Иконки автоматически загрузятся
4. Переместите их в папку `MacThree-Web`

## Решение 3: Игнорировать ошибки (временно)

Игра работает и без иконок. Ошибки 404 не влияют на функциональность.

## Что уже исправлено

- ✅ Добавлен favicon через data URI (не требует файла)
- ✅ Ссылка на apple-touch-icon сделана опциональной (onerror скрывает ошибку)
- ✅ Создан скрипт для генерации иконок

## После создания иконок

Просто обновите страницу в браузере (F5). Ошибки 404 исчезнут.
