# MacThree Web - PWA версия для iPhone

Веб-версия игры Match-3, которую можно установить на iPhone как приложение (PWA).

## Особенности

✅ **Работает на Windows** - можно разрабатывать и тестировать на любом компьютере  
✅ **Установка на iPhone** - можно добавить на главный экран как приложение  
✅ **Офлайн режим** - работает без интернета после первой загрузки  
✅ **Полная функциональность** - все возможности оригинальной игры  
✅ **Адаптивный дизайн** - оптимизировано для iPhone  

## Как использовать

### 1. На компьютере (разработка/тестирование)

1. Откройте `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Или Node.js
   npx http-server
   ```
3. Откройте `http://localhost:8000` в браузере

### 2. На iPhone (установка как приложение)

1. **Загрузите файлы на веб-сервер:**
   - Загрузите всю папку `MacThree-Web` на любой веб-хостинг
   - Или используйте GitHub Pages, Netlify, Vercel и т.д.

2. **Откройте в Safari на iPhone:**
   - Откройте сайт в Safari (не в Chrome!)
   - Нажмите кнопку "Поделиться" (квадрат со стрелкой)
   - Выберите "На экран «Домой»"
   - Нажмите "Добавить"

3. **Готово!** Игра появится на главном экране как приложение

## Создание иконок

Для полноценной работы PWA нужны иконки:

1. Создайте иконку 512x512 пикселей
2. Сохраните как `icon-512.png`
3. Создайте уменьшенную версию 192x192 как `icon-192.png`
4. Или используйте онлайн-генератор иконок

## Функции

- ✅ Черный фон главного меню
- ✅ Поле ввода логина на главной странице
- ✅ Игровая доска 8x8
- ✅ Поиск совпадений по горизонтали, вертикали и диагонали
- ✅ Система очков (5 очков за каждую фишку)
- ✅ Достижения (каждые 500 очков)
- ✅ Таблица рекордов (сохранение в LocalStorage)
- ✅ Адаптивный дизайн для разных размеров экрана
- ✅ Поддержка сенсорного управления

## Развертывание

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы
3. Включите GitHub Pages в настройках репозитория
4. Откройте `https://ваш-username.github.io/репозиторий/`

### Netlify

1. Зарегистрируйтесь на netlify.com
2. Перетащите папку `MacThree-Web` в Netlify
3. Получите URL для доступа

### Vercel

1. Установите Vercel CLI: `npm i -g vercel`
2. В папке проекта выполните: `vercel`
3. Следуйте инструкциям

## Решение проблем

### Игра не устанавливается на iPhone
- Убедитесь, что используете Safari (не Chrome)
- Проверьте, что сайт открыт по HTTPS (обязательно для PWA)
- Убедитесь, что `manifest.json` доступен

### Изображения не загружаются
- Проверьте пути к файлам
- Убедитесь, что все файлы загружены на сервер

### Рекорды не сохраняются
- Проверьте, что LocalStorage доступен в браузере
- Убедитесь, что не используете режим инкогнито
