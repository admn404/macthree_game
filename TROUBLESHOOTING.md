# Решение проблем

## Проблема: При нажатии "Новая игра" ничего не происходит или ошибка

### Решение 1: Проверьте консоль браузера

1. Откройте консоль разработчика:
   - **Chrome/Edge**: F12 или Ctrl+Shift+I
   - **Firefox**: F12 или Ctrl+Shift+K
   - **Safari**: Cmd+Option+I (Mac)

2. Посмотрите, какие ошибки выводятся в консоли

3. Скопируйте текст ошибки и проверьте ниже

### Решение 2: Проверьте, что все файлы загружены

Убедитесь, что в консоли нет ошибок 404 для:
- `game.js`
- `styles.css`
- `index.html`

### Решение 3: Очистите кэш браузера

1. Нажмите Ctrl+Shift+Delete
2. Выберите "Кэшированные изображения и файлы"
3. Нажмите "Очистить данные"
4. Обновите страницу (F5)

### Решение 4: Проверьте версию браузера

Игра требует современный браузер с поддержкой:
- HTML5 Canvas
- ES6 JavaScript
- LocalStorage

Рекомендуемые браузеры:
- Chrome 90+
- Firefox 88+
- Edge 90+
- Safari 14+

### Решение 5: Проверьте JavaScript

Откройте консоль и выполните:
```javascript
console.log(typeof canvas);
console.log(typeof ctx);
console.log(gameState);
```

Если что-то undefined, значит есть проблема с инициализацией.

## Частые ошибки

### "Canvas элемент не найден"
- Убедитесь, что `index.html` содержит `<canvas id="game-canvas"></canvas>`
- Проверьте, что скрипт загружается после создания canvas

### "Cannot read property 'getContext' of null"
- Canvas не найден при инициализации
- Проверьте порядок загрузки скриптов

### "drawBoard is not a function"
- Файл `game.js` не загружен полностью
- Проверьте, что файл доступен по URL

### Игра не реагирует на клики
- Проверьте, что canvas видим (не скрыт)
- Убедитесь, что обработчики событий установлены

## Отладка

Добавьте в начало `game.js`:
```javascript
console.log('Game.js загружен');
console.log('Canvas:', canvas);
console.log('Context:', ctx);
```

Это поможет понять, на каком этапе возникает проблема.
